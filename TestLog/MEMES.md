# Як замінити меми на свої картинки

Гра використовує **50 локальних мем-карток**, зашитих прямо в APK
(`assets/memes/meme_01.png` … `meme_50.png`). Завдяки цьому меми працюють
**повністю офлайн** — інтернет не потрібен.

Є два способи замінити їх на свої.

---

## Спосіб 1 (найпростіший): підмінити файли

1. Підготуй свої картинки у форматі **PNG**, бажано квадратні (напр. 512×512).
2. Назви їх рівно так: `meme_01.png`, `meme_02.png`, … `meme_50.png`.
3. Поклади їх у папку `assets/memes/`, замінивши наявні.
4. Перезбери APK (див. нижче «Збірка»).

> Кількість карток має лишатись **50** (id 1…50). Якщо хочеш іншу кількість —
> дивись «Зміна кількості карток».

---

## Спосіб 2: свої меми через генератор (емодзі + підпис)

У репозиторії є скрипт `scripts/gen_memes.py`, який малює картки
(градієнт + велике емодзі + підпис українською).

1. Відкрий `scripts/gen_memes.py`, знайди список `CARDS`.
2. Кожен рядок — `(емодзі, "підпис", колір1, колір2)`. Зміни на свої.
3. Запусти:
   ```bash
   pip install Pillow
   python3 scripts/gen_memes.py
   ```
   Скрипт перезапише `assets/memes/meme_01..50.png`.
4. Перезбери APK.

---

## Якщо хочеш фотографії-меми (з підписами зверху/знизу)

Це класичний формат «impact-меми». Можна:
- намалювати їх у будь-якому редакторі та зберегти як `meme_NN.png`, **або**
- розширити `scripts/gen_memes.py`: замість градієнта завантажувати базову
  картинку й накладати білий текст із чорним обведенням (Pillow:
  `ImageDraw.text(..., stroke_width=3, stroke_fill="black")`).

---

## Зміна кількості карток (не обов'язково)

Картки описані у двох місцях:

1. **`src/game/engine/deck.ts`** — масив `MEME_CARDS` (id + підпис `title`).
   Додай/прибери записи (id мають бути послідовні 1…N).
2. **`src/game/engine/memeAssets.ts`** — таблиця `MEME_ASSETS`, що звʼязує
   `id → require('../../../assets/memes/meme_NN.png')`.
   Metro вимагає **статичні** літерали в `require()`, тому кожен файл
   прописується окремим рядком (не можна зібрати шлях динамічно).

> Рекомендація: лишай ≥ 50 карток, щоб на 3–8 гравців вистачало роздачі
> (8 карт на старті + добір щораунду).

---

## Як роздаються картки (нова механіка)

- На старті кожен гравець отримує **8 карт** зі **спільної колоди**.
- Кожного раунду гравець добирає **1 карту** (зіграна йде у скид).
- Коли колода вичерпується — скид перемішується назад.
- Тобто колода **спільна на всю гру**, а не нова щораунду.

(Логіка — у `src/game/engine/engine.ts`: `initDeck`, `drawCard`, `dealCards`.)

---

## Збірка APK

```bash
cd /шлях/до/TestLog
yarn install            # або npm install
# Android SDK + JDK 17 мають бути встановлені, ANDROID_HOME заданий
cd android
./gradlew :app:assembleRelease -PreactNativeArchitectures=arm64-v8a
```

Готовий файл: `android/app/build/outputs/apk/release/app-release.apk`.

> Підпис — debug-ключем, тож APK одразу ставиться на телефон.
> Картинки з `assets/memes/` автоматично потрапляють у APK при збірці.
